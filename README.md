# E-Commerce Sales Analysis for Data-Driven Decision Making

## Project Overview
This project analyzes e-commerce sales data to understand customer behavior, sales trends, product performance, and business insights.

## Tasks Completed
- Market research on Indian e-commerce industry
- Dataset exploration
- Data cleaning and preprocessing

## Dataset
Global Superstore Dataset from Kaggle

## Tools Used
- Python
- Pandas
- Matplotlib
- VS code

## Files
- superstore.csv → raw dataset
- final_cleaned_superstore.csv → cleaned dataset
- E_Commerce_Analysis.ipynb → analysis notebook
